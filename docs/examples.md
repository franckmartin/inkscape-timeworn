# Examples

## Weathered Fresco Effect

Perfect for simulating ancient wall paintings with natural paint deterioration.

**Parameters:**

- Size: 0.5-2.5mm
- Density: 500-1000
- Clustering: 60-80%
- Number of clusters: 8-12
- Irregularity: 40-60%
- Elongation factor: 2.0-3.0
- Angle variation: 30-50%

**Best for:** Historical artwork, aged posters, vintage graphics

---

## Tips for Custom Effects

!!! tip "Layering"
    Apply the extension multiple times with different parameters for complex, multi-layered weathering.

!!! tip "Color Variations"
    After generating spots, use Inkscape's color tools to create rust, patina, or other colored weathering effects.

!!! tip "Masking"
    Use different boundary shapes to create selective weathering in specific areas.

!!! tip "Blur & Opacity"
    Combine with Inkscape's blur filter and opacity settings for softer, more subtle effects.
